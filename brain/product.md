Product Definition

Foldaa is a programmable application runtime.

The platform allows users to define software systems through structured definitions rather than traditional programming.

Applications inside Foldaa consist of:

Models
Workflows
Nodes
Deployments

These definitions are interpreted by the Foldaa runtime.

Capabilities:

• Automatic API generation
• Workflow orchestration
• Tool execution
• Infrastructure deployment
• Domain management
• Event driven execution

Foldaa is intended to support multiple types of builders:

Developers
Design engineers
Product teams
AI agents

The product focuses on enabling systems where logic and infrastructure are defined declaratively.

Foldaa is not intended to replace frontend frameworks.

Instead it provides the programmable backend runtime for them.
