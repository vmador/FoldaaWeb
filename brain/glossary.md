Glossary

Contract
Structured data definition representing application entities.

Workflow
Declarative logic responding to events.

Node
Executable tool used within workflows.

Event
Trigger that initiates workflow execution.

Runtime
The engine responsible for interpreting application definitions and executing them.

Deployments: Infrastructure resources required to run applications.

Campfire
The orchestration layer and lifecycle for Temps and Tents.

Temp
A pre-configured development kit (mini-Tent) rented by clients.

Flame
An active feature branch, sprint, or development task within a Tent.

Tent
A living project instance ignited from a Temp where Builders collaborate.

Builder
A specialized contributor hired for a specific duration/capacity within a Tent.

Brain
The AI knowledge layer that holds context, analyzes SoT, and manages transferability.

Igniter
The client or project owner who selects and rents a Temp.

Source of Truth (SoT)
The universal documentation manifest (`system.md`, `design.md`, `roadmap.md`, `stack.md`).
